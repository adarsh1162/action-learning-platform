# Module 4 Quiz

### Q1

Analyze the code:

```javascript
let finalValue = "10" + 5 - 2;
```

Which statements are true?

* A. `"10" + 5` produces `"105"`.
* B. Final value becomes `103`.
* C. Final value becomes `"103"`.
* D. JavaScript performs concatenation before subtraction here.
* E. The code throws an error.

**Correct Answers:** A, B, D

---

### Q2

Analyze the code:

```javascript
let result = 20 / 4 * 2;
```

Which statements are true?

* A. Division executes before multiplication because `/` has higher precedence.
* B. Multiplication and division have equal precedence.
* C. Expression is evaluated left to right.
* D. Final value is `10`.
* E. Final value is `2.5`.

**Correct Answers:** B, C, D

### Q3

```javascript
let totalItems = 3;
let containerCapacity = 10;
```

Which statements are true?

* A. `remainder` becomes `0`.
* B. `remainder` becomes `3`.
* C. `%` returns the remainder.
* D. The first number can be smaller than the second.
* E. The code throws an error.

### Q7

Analyze the code:

```javascript

```

* A. `a` becomes `6`.
* B. `b` becomes `6`.
* C. `b` receives the original value first.
* D. Final values are `a = 6` and `b = 5`.

### Q8


* C. This is prefix increment behavior.

### Q10

An AI leaderboard system contains the following fixed code:

```javascript
let baseScore = 500;
let streakBonus = 20;
let multiplier = 1.5;

let finalScore =
    (baseScore + streakBonus) *
    multiplier;

let userTokens = 1000;
let apiCost = 45.9;

userTokens -= Math.floor(apiCost);

let loginDays = 14;
loginDays++;

let serverShard =
    finalScore % 3;

let challengeId =
    Math.floor(Math.random() * 10);
```

Which statements are true?

* A. `finalScore` evaluates to `780`.
* B. `userTokens` evaluates to `955`.
* C. `loginDays` becomes `15`.
* D. `serverShard` evaluates to `0`.
* E. `challengeId` is guaranteed to be a whole number between `0` and `9`.

**Correct Answers:** A, B, C, D, E
