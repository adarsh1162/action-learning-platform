# Module 3 Quiz

### Q1

Analyze the code:

```javascript
let price = 100;
let taxRate = 0.18;
```

Which statements are true?

* A. `price` is a number.
* B. `taxRate` is a number.
* C. JavaScript stores integers and decimals using different primitive types.
* D. `typeof price` returns `"number"`.
* E. `typeof taxRate` returns `"number"`.

**Correct Answers:** A, B, D, E

---

### Q2

Analyze the code:

```javascript
let result = "Apple" * 5;
```

Which statements are true?

* A. `result` becomes `NaN`.
* B. `typeof result` returns `"number"`.
* C. `result === NaN` evaluates to true.
* D. `Number.isNaN(result)` evaluates to true.
* E. The code throws a syntax error.

**Correct Answers:** A, B, D

### Q3

```javascript

```

Which statements are true?

* A. Output is `true`.
* B. Output is `false`.
* C. `NaN` is equal to itself.
* D. `Number.isNaN(errorValue)` is the correct way to test it.
* E. `NaN` is a special numeric value.

### Q10

A banking system contains the following code:

```javascript
let applicantName = "Sarah O'Connor";
let annualIncome = 85000;
let taxMultiplier = 5;

let calculatedTax =
    annualIncome / taxMultiplier;

let isIdentityVerified = true;

let uploadedDocuments = null;

let taxTypeCheck =
    typeof calculatedTax;

let finalLog =
`Applicant: ${applicantName}
Tax: ${calculatedTax}
Verified: ${isIdentityVerified}`;
```

Which statements are true?

* A. `calculatedTax` evaluates to `17000`.
* B. `Number.isNaN(calculatedTax)` returns `false`.
* C. `isIdentityVerified` stores a boolean.
* D. `taxTypeCheck` equals `"number"`.
* E. The final log correctly uses a template literal.

**Correct Answers:** A, B, C, D, E
