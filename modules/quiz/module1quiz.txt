# Module 1 Quiz

### Q1

Analyze the code:

```javascript
console.log("System Start");
console.warn("High CPU");
console.error("System Crash");
```

Which statements are true?

* A. The warning executes after the normal log.
* B. The error executes before the warning.
* C. Three messages are printed.
* D. JavaScript executes the statements from top to bottom.
* E. `console.error()` stops the program automatically.

**Correct Answers:** A, C, D

---

### Q2

What will be printed?

```javascript
let health = 100;

console.log(health);

health = 50;

console.log(health);
```

* A. First output is `100`
* B. Second output is `50`
* C. Both outputs are `100`
* D. Reassignment changes the value stored in memory
* E. The second log runs before the reassignment

**Correct Answers:** A, B, D

---

### Q3

Analyze the code:

```javascript
let score = 10;

console.log(score);

score = score + 5;

console.log(Score);
```

Which statements are true?

### Q7

### Q9

Analyze the code:

```javascript
let result = if (10 > 5) {
    true;
} else {
    false;
};
```

Why is this invalid?

* A. `if/else` is a statement, not an expression.
* B. The assignment operator expects a value on the right side.
* C. JavaScript allows statements to be assigned directly.
* D. The code causes a syntax error.
* E. `if/else` automatically returns a value.

**Correct Answers:** A, B, D

### Q10

A developer must satisfy these requirements:

1. Check power supply first.
2. Connect to database second.
3. Report database failure third.
4. Report diagnostics last.
5. Power issue must be a warning.
6. Database failure must be an error.

Which implementation satisfies all requirements?

```javascript
Option A

console.log("Server reboot initiated.");
console.warn("Power supply unstable.");
console.log("Connecting to main database...");
console.error("Database connection failed.");
console.log("System diagnostic complete");
```

```javascript
Option B

console.log("Server reboot initiated.");
console.error("Power supply unstable.");
console.log("Connecting to main database...");
console.warn("Database connection failed.");
console.log("System diagnostic complete");
```

```javascript
Option C

console.log("Server reboot initiated.");
console.log("Connecting to main database...");
console.warn("Power supply unstable.");
console.error("Database connection failed.");
console.log("System diagnostic complete");
```

* A. Option A satisfies all requirements.
* B. Option B satisfies all requirements.
* C. Option C satisfies all requirements.
* D. Correct console types matter as much as correct order.
* E. Changing warning/error methods can break the requirements.

**Correct Answers:** A, D, E
