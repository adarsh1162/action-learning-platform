# Module 5 Quiz

### Q1

Analyze the code:

```javascript
let score = "100";

let converted =
    Number(score);
```

Which statements are true?

* A. `converted` becomes the number `100`.
* B. `typeof converted` is `"number"`.
* C. `converted` remains a string.
* D. `Number()` performs explicit conversion.
* E. `typeof converted` is `"string"`.

**Correct Answers:** A, B, D

---

### Q10

An e-commerce checkout system contains the following code:

```javascript
let productPrice = "120";
let shippingTax = "15";
let userPoints = "0";
let hasVoucher = false;
let couponInput = "";

userPoints =
    Number(userPoints);

let cartTotal =
    Number(productPrice) +
    Number(shippingTax);

if (couponInput) {
    console.log("Coupon Applied");
}

if (userPoints) {
    console.log("Using Points");
}
```

Which statements are true?

* A. `cartTotal` evaluates to `135`.
* B. `cartTotal` is a number.
* C. `"Coupon Applied"` is never logged.
* D. `"Using Points"` is never logged.
* E. The code avoids the major coercion bugs from the original checkout system.

**Correct Answers:** A, B, C, D, E
