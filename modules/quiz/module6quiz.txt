### Q1

Analyze the code:

```javascript
let currentLevel = "10";
let nextLevel = "2";

console.log(
    currentLevel > nextLevel
);
```

Which statements are true?

* A. The expression evaluates to `true`.
* B. The expression evaluates to `false`.
* C. JavaScript compares the strings character by character.
* D. `"1"` is considered smaller than `"2"`.
* E. The values are automatically converted to numbers before comparison.

**Correct Answers:** B, C, D

### Q10

An anti-cheat system contains the following corrected logic:

```javascript

let cheatFlags =
    Number("0");

let isProctorActive = false;

if (
    userScore > 90 &&
    !isProctorActive

    console.log("Locked");
}

    console.log("Grandmaster");
}
else if (userScore >= 75) {
    console.log("Silver");
}
else if (userScore >= 50) {
    console.log("Bronze");
}

    25 < 30
    ? "Approved"
    : "Denied";
```

* A. The account is locked.
* B. The tier becomes `"Grandmaster"`.

* E. The code fixes the major logic flaws from the original anti-cheat engine.
