# 🟢 EASY 1 — The Missing Property

### Mission Brief

A developer wrote the following code:

```js
function getSecret(user) {
    return user.secretKey.toUpperCase();
}
```

Find an input that crashes the function.

### Solution

```js
{}
```

### Why It Breaks

The code assumes every object has a `secretKey` property.

```js
{}.secretKey
```

returns:

```js
undefined
```

and:

```js
undefined.toUpperCase()
```

throws a TypeError.

### Secure Fix

```js
function getSecret(user) {
    return user.secretKey?.toUpperCase() ?? "No Key";
}
```

# 🟡 INTERMEDIATE 13 — Infinite Coupon

```js
function applyCoupons(price, coupons) {
    let i = 0;

    while(i < coupons.length) {

        price -= coupons[i].value;

        if(coupons[i].expired)
            continue;

        i++;
    }

    return price;
}
```

### Solution

```js
[
    {
        value: 10,
        expired: true
    }
]
```

### Why It Breaks

`continue` skips `i++`.

The loop stays forever at index 0.

# 🟡 INTERMEDIATE 15 — Recursive Nightmare

```js
function countdown(n) {

    if(n === 0)
        return;

    countdown(n - 1);
}
```

### Solution

```js
Infinity
```

### Why It Breaks

```js
Infinity - 1 === Infinity
```

The recursion never reaches zero.

Eventually:

```js
Maximum call stack size exceeded
```

# 🔴 HARD 19 — Self Referencing Loop

### Mission Brief

```js
function stringify(data) {
    return JSON.stringify(data);
}
```

### Solution

```js
const obj = {};
obj.self = obj;
```

### Why It Breaks

The object contains a circular reference.

`JSON.stringify()` cannot serialize circular structures.
